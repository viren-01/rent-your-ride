Welcome to our Ride Booking Website, your ultimate destination for booking cars and rides with ease and convenience. Our platform offers a wide range of vehicle selections to cater to all your transportation needs, whether it's for a business trip, a family vacation, or just a casual ride around town. We pride ourselves on offering competitive prices that make transportation affordable for everyone.

#### Features

- **Extensive Vehicle Selection**: Choose from a variety of vehicles, including sedans, SUVs, hatchbacks, and more, to find the perfect ride for your journey.

- **Affordable Prices**: Enjoy some of the best rates in the market without compromising on quality or comfort.

- **User-Friendly Interface**: Our intuitive and easy-to-navigate website ensures a seamless booking experience.

- **Secure Booking**: Your safety and privacy are our top priorities, with secure payment options and data protection measures in place.

- **24/7 Customer Support**: Get assistance whenever you need it with our round-the-clock customer support.

Whether you're planning a long trip or need a quick ride, our website is designed to make your booking process as smooth and hassle-free as possible. Explore our selection and book your ride today!

## Run Locally

Clone the project

```bash
  git clone https://github.com/viren-01/rent-your-ride
```
#### Set up Frontend & Backend
Go to the project directory

```bash
  cd frontend (for backend run cd backend)
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Environment Variables

To run this project, you will need to add the following environment variables to your file

Frontend

`REACT_APP_BACKEND_URL`

Backend

`DB_HOST`
`DB_USER`
`DB_PASS`
`DB_NAME`
`DB_PORT`
`HOST`
`PORT`
`SMTP_USER`
`SMTP_PASS`
